# HabitForge

A Gamified Habit Tracking Application.

## Features

- Add / Delete Habits
- Habit Streak Tracking
- XP + Levels
- Progress Bar
- Achievement Badges
- Analytics Chart
- MongoDB Storage

## Tech Stack

- Node.js
- Express.js
- MongoDB
- HTML/CSS/JavaScript
- Chart.js

## How to Run

npm install

node final.js
